## 🚀 在群晖NAS上部署

### 最简单方法（推荐）

#### 步骤1: 准备群晖
1. 确保安装了Container Manager
2. 确保安装了Docker Compose

#### 步骤2: 上传文件
使用File Station上传文件到 `/docker/rental-system`:
- `Dockerfile`
- `docker-compose.yml`
- `package.json`
- `server-sqlite.js`
- `public/` 文件夹

#### 步骤3: 启动服务
使用Container Manager图形界面：
1. 打开Container Manager
2. 容器 → 创建
3. 选择"从docker-compose文件创建"
4. 选择 `docker-compose.yml`
5. 点击"创建"

或使用SSH命令：
```bash
cd /docker/rental-system
./一键部署.sh
```

#### 步骤4: 访问系统
打开浏览器：
```
http://192.168.1.5:4000
```

登录凭据：
- 用户名: `admin`
- 密码: `admin123`

## 🔧 常用操作

### 查看状态
```bash
docker-compose ps
```

### 查看日志
```bash
docker-compose logs -f
```

### 重启服务
```bash
docker-compose restart
```

### 停止服务
```bash
docker-compose down
```

### 修改端口
编辑 `docker-compose.yml`:
```yaml
ports:
  - "8080:4000"
```

## 📝 数据管理

### 数据位置
- 数据库文件: `/docker/rental-system/data/rental_system.db`

### 备份数据
```bash
cp /docker/rental-system/data/rental_system.db /backup/
```

### 恢复数据
```bash
cp /backup/rental_system.db /docker/rental-system/data/
```
