## 🚀 在macOS上使用

### 1. 克隆仓库

```bash
cd ~/Documents
git clone https://github.com/kavaman84/rental-management-system.git
cd rental-management-system
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动服务器

```bash
npm start
```

### 4. 访问系统

打开浏览器访问：
```
http://localhost:3000
```
