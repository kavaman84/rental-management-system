# 快速入门

## 第一次使用

1. **安装MySQL并创建数据库**
   ```bash
   mysql -u root -p < schema.sql
   ```

2. **配置数据库连接**
   编辑 `server.js`，设置正确的MySQL密码

3. **启动服务器**
   ```bash
   npm start
   ```

4. **访问系统**
   打开浏览器访问: http://localhost:3000

5. **登录**
   - 用户名: `admin`
   - 密码: `admin123`

## 使用流程

### 添加新房间

1. 登录系统
2. 点击"管理房间"
3. 点击"详情"查看或修改房间信息
4. 修改月租金、税费单价、电费单价、水费单价
5. 点击"保存修改"

### 记录电表水表读数

1. 在房间详情页面
2. 电表水表读数会自动显示在"电表水表读数记录"表格中
3. 如果需要添加新读数，需要直接操作数据库或修改代码

### 生成月收据

1. 点击"收据管理"
2. 选择房间
3. 选择收据月份
4. 点击"生成收据"
5. 系统会自动：
   - 读取上个月的电表水表读数
   - 计算用电量、用水量
   - 根据费率计算各项费用
   - 生成收据

### 标记收据为已支付

1. 在收据列表中找到未支付的收据
2. 点击"支付"按钮
3. 确认后状态变为"已支付"

## 数据结构

### 房间
- 房号（唯一）
- 月租金
- 税费单价（每元租金的税费）
- 电费单价（元/度）
- 水费单价（元/吨）

### 收据
- 房间
- 收据月份
- 月租金
- 税费金额
- 电费金额
- 水费金额
- 总金额
- 用电量
- 用水量
- 状态（未支付/已支付）
