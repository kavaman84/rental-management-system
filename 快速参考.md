# 快速参考卡

## 🚀 快速部署

### 在群晖上执行：

```bash
cd /docker/rental-system
./一键部署.sh
```

### 访问系统：

```
http://192.168.1.5:4000
```

**登录**: admin / admin123

## 📝 常用命令

```bash
# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 重启
docker-compose restart

# 停止
docker-compose down

# 启动
docker-compose up -d
```

## 📂 文件说明

- `Dockerfile` - Docker镜像构建文件
- `docker-compose.yml` - 容器编排配置
- `一键部署.sh` - 一键部署脚本（推荐）
- `deploy.sh` - 部署脚本
- `start.sh` - 启动脚本
- `stop.sh` - 停止脚本

## ⚙️ 配置

### 修改端口

编辑 `docker-compose.yml`:
```yaml
ports:
  - "8080:4000"  # 修改群晖端口
```

### 修改内存限制

编辑 `docker-compose.yml`:
```yaml
deploy:
  resources:
    limits:
      memory: 1024M
```

## 🔧 故障排查

### 无法访问

1. 检查容器: `docker ps`
2. 检查日志: `docker-compose logs`
3. 重启: `docker-compose restart`

### 端口被占用

修改 `docker-compose.yml` 中的端口映射
